# SpotReview (스팟리뷰)

> **네이버 지도 API 기반의 주요 명소 탐색 및 실시간 리뷰 공유 안드로이드 애플리케이션**
>
> 사용자는 지도 상에서 원하는 명소를 직관적으로 탐색하고, 다른 사용자들이 남긴 생생한 리뷰와 순위 정보를 확인 및 공유할 수 있습니다. 관리자 기능을 통해 안전하고 깨끗한 커뮤니티 환경을 유지합니다.

---

## 주요 구현 기능

### 1. 회원 관리 (User Management)
* **회원가입 & 로그인**: 사용자 계정 생성 및 안전한 인증 시스템 구현
* **ID/PW 찾기**: 계정 정보를 분실했을 경우를 대비한 사용자 식별 및 비밀번호 복구 기능

### 2. 메인 대시보드 (Main Menu)
* **주요 스팟 탐색 이동**: 한 번의 클릭으로 지도 기반 탐색 화면으로 전환
* **통계 및 랭킹 시스템**:
    * 등록된 명소별 총 리뷰 개수 실시간 집계
    * 리뷰 개수를 기반으로 한 **명소 인기 순위(랭킹) 출력** 구현

### 3. 지도 기반 주요 스팟 탐색 (Map & Review Service)
* **네이버 지도 API & 검색 API 연동**:
    * 네이버 지도를 앱 내에 구현하고 검색 API를 통해 정확한 명소 위치 데이터 확보
    * 검색된 주요 명소 위치에 **커스텀 마커(Marker) 출력**
* **상호작용 이벤트**:
    * 마커 클릭 시 해당 스팟의 상세 정보를 보여주는 **이벤트 리스너** 구현
    * 마커 클릭 시 즉시 리뷰를 확인하고 작성할 수 있는 **리뷰 창(Bottom Sheet/Dialog) 오픈**
* **리뷰 작성 및 조회**:
    * 명소 이름, 작성자 유저 이름, 리뷰 내용 입력 및 실시간 리스트 출력 기능 구현

### 4. 관리자 시스템 (Admin Features)
* **콘텐츠 관리**: 부적절하거나 스팸성 리뷰를 관리자가 직접 **삭제 및 복원**할 수 있는 기능
* **유저 관리**: 커뮤니티 가이드를 위반한 악성 **유저 차단 및 복원(차단 해제)** 기능 구현

## 기술 스택 (Tech Stack)

* **Platform**: Android (Java / OpenJDK 23)
* **IDE**: Android Studio
* **Build Tool**: Gradle / Maven
* **UI/UX**: Material Design Components
* **External APIs**:
    * Naver Map API
    * Naver Search API
